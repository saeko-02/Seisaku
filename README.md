# 商品管理システム

## 概要
このシステムでは、商品の在庫管理を行うことができます。

商品の管理一覧画面から、商品の編集・削除を行うことができます。

## 主な機能
-ログイン・ログアウト機能
-商品一覧画面
-商品編集画面
-商品編集・削除機能

## 開発環境

PHP 8.2.12
MySQL 
Laravel 10.13.5
