# 日用品管理システム

## 概要
このシステムでは、日用品の在庫管理を行うことができます。

日用品の管理一覧画面から、日用品の編集・削除、検索を行うことができます。

## 主な機能
-ログイン・ログアウト機能
-日用品一覧画面
-日用品編集画面
-日用品編集・削除機能
-検索機能
